Belajar bikin paragraf di html


Tag p

Tag yang bisa di gunakan yaitu tag (P)

contoh tampilan nya seperti yang anda baca sekarang ini

Tag Strong dan em

Tag untuk mencetak tulisan tebal atau di beri penekanan bisa menggunakan tag strong atau membuat tulisan miring bisa menggunakan tag em

Tag tag b, i, dan u

Fungsi tag b untuk membuat teks menjadi bold
Fungsi tag i untuk membuat teks menjadi italic
Fungsi tag u untuk membuat teks menjadi underline

tag ini bisa di gunakan sebagai dekorasi , mengatur teks bisa juga menggunakan css

Tag hr dan br

Tag untuk membuat garis horizontal bisa menggunakan tag hr dan untuk membuat enter atau paragraf baru bisa menggunakan tag br
